# Musik latar

Letakkan file musik berlisensi/berizin di folder ini, misalnya `birthday-song.mp3`.
Kemudian isi `musicUrl` melalui sheet `Config` atau fallback di `frontend/js/config.js`.

Rekomendasi:
- MP3/AAC 96–128 kbps;
- durasi singkat atau file di bawah 3 MB;
- pastikan hak penggunaan musik telah dimiliki.
