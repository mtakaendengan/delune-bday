# Attribution

Proyek ini merupakan implementasi ulang untuk undangan ulang tahun digital.
Pola UX desktop split-panel, cover pembuka, section kartu, dan navigasi bawah
mengikuti kebutuhan yang disimpulkan dari template referensi yang diberikan
pengguna. Kode, struktur backend Apps Script, CSS, SVG placeholder, dan
JavaScript pada paket ini ditulis ulang. Media audio/video dari arsip referensi
tidak disertakan.
